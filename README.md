# Minesweeper Grid Editor

A simple, fully functional web-based tool to create and edit **custom Minesweeper grids**.  
Perfect for puzzle creators, game developers, or anyone who wants to design their own Minesweeper boards.

![Minesweeper Grid Editor Preview](https://via.placeholder.com/800x400?text=Minesweeper+Grid+Editor+Preview)

## ✨ Features

- **Resizable grid** — Choose any size from 5×5 up to 30×40
- **Click to toggle bombs** — Intuitive editing
- **Real-time number calculation** with classic Minesweeper colors
- **Flexible Export Options**:
  - Text (space-separated) or CSV format
  - Bombs Only or Bombs + Solution (numbers)
- **Import support** — Paste back any previously exported board (auto-resizes grid if needed)
- **Randomize** button (~15% bomb density)
- **Clear** board with one click
- Works offline — just open the HTML file in any browser

## How to Use

1. Download the repository
2. Open `minesweeper-editor.html` in your browser (Chrome, Edge, Firefox, or Safari)
3. Start creating your Minesweeper puzzle!

### Controls
- **Rows / Columns** + **Create New Grid** → Change board size
- Click any cell → Toggle bomb on/off
- **Randomize** → Fill with ~15% random bombs
- **Export** → Choose format and content, then copy to clipboard
- **Import** → Paste any previously exported board


## Export Formats Explained

| Format       | Include Solution | Result Example                  | Best For                  |
|--------------|------------------|----------------------------------|---------------------------|
| Text         | No               | `* . * .`                        | Human readable puzzles    |
| Text         | Yes              | `* 1 * 2`                        | Full solution sharing     |
| CSV          | No               | `*,.,*,.,`                       | Excel / Google Sheets     |
| CSV          | Yes              | `*,1,*,2,`                       | Data analysis / import    |
